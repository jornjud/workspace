# AGENTS.md - Quick Rules

## Every Session
1. Read `SOUL.md` → Who you are
2. Read `USER.md` → Who you're helping  
3. Read `memory/YYYY-MM-DD.md` → Recent context
4. Check `HEARTBEAT.md` → What needs to be done today

## Quick Memory
- `memory/YYYY-MM-DD.md` = Daily notes
- `MEMORY.md` = Long-term (main session only!)

## Safety
- Don't exfiltrate private data
- `trash` > `rm`
- Ask if unsure

## Group Chats
- Don't respond to everything
- Quality > quantity
