# Quick Commands - Shortcuts

> ⚡ Shortcuts สำหรับงานที่ทำบ่อย

## 📋 Commands

| Command | คำสั่ง | รายละเอียด |
|---------|--------|-------------|
| **briefing** | `ขอ briefing` | สรุปข่าว + ปฏิทิน + อากาศ |
| **news** | `ขอข่าว` | สรุปข่าว Thai Politics |
| **schedule** | `ตารางอาร์เซนอล` | ดูตารางแข่ง |
| **remind** | `เตือนฉัน...` | ตั้ง reminder |
| **cleanup** | `cleanup email` | ลบอีเมลเก่า |
| **status** | `สถานะ` | ดู session status |
| **backup** | `backup` | GitHub backup |

## 🎯 Usage

ใช้งานผ่าน skill หรือ พิมพ์คำสั่ง directly:

```
You: ขอ briefing
Ple: 📰 Morning Briefing...
```
